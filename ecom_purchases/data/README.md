# Dataset

**File:** `Ecommerce Purchases.csv`

Synthetic e-commerce purchase data from Jose Portilla's *Python for Data Science and Machine Learning Bootcamp* (~10,000 rows, 14 columns).

**Expected columns include:** `Purchase Price`, `AM or PM`, `Language`, `Job`, `Lot`, `Credit Card`, `Email`

**Re-download (if needed):**

```bash
curl -L -o "Ecommerce Purchases.csv" \
  "https://raw.githubusercontent.com/reza16977/Ecommerce-Purchases-Exercise/main/Ecommerce%20Purchases"
```

Note: the upstream file has no `.csv` extension; save it as `Ecommerce Purchases.csv`.
