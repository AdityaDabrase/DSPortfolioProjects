# Dataset

**File:** `Ecommerce Purchases.csv`

**Download into this folder:**

```bash
curl -L -o "Ecommerce Purchases.csv" \
  "https://raw.githubusercontent.com/reza16977/Ecommerce-Purchases-Exercise/main/Ecommerce%20Purchases"
```

Expected columns include `Purchase Price`, `AM or PM`, `Language`, `Job`, `Credit Card`, `Email`.
