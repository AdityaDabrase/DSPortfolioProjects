# Dataset

**File:** `advertising.csv`

Ad-click dataset from Jose Portilla's *Python for Data Science and Machine Learning Bootcamp*.

This is **not** the TV/Radio/Newspaper Kaggle advertising file. Expected columns:

`Daily Time Spent on Site`, `Age`, `Area Income`, `Daily Internet Usage`, `Ad Topic Line`, `City`, `Male`, `Country`, `Timestamp`, `Clicked on Ad`

**Re-download (if needed):**

```bash
curl -L -o advertising.csv \
  "https://raw.githubusercontent.com/anishapareek/ad-clicks-prediction/main/advertising.csv"
```
