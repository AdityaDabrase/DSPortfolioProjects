# Dataset

**File:** `advertising.csv`

Ad-click dataset (not the TV/Radio/Newspaper advertising file).

**Download into this folder:**

```bash
curl -L -o advertising.csv \
  "https://raw.githubusercontent.com/anishapareek/ad-clicks-prediction/main/advertising.csv"
```

Expected columns: `Daily Time Spent on Site`, `Age`, `Area Income`, `Daily Internet Usage`, `Male`, `Clicked on Ad`, etc.
