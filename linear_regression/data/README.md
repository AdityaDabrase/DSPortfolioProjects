# Dataset

**File:** `Ecommerce Customers.csv`

**Download into this folder:**

```bash
curl -L -o "Ecommerce Customers.csv" \
  "https://raw.githubusercontent.com/araj2/customer-database/master/Ecommerce%20Customers.csv"
```

Expected columns include `Avg. Session Length`, `Time on App`, `Time on Website`, `Length of Membership`, `Yearly Amount Spent`.
