# Dataset

**File:** `Mall_Customers.csv`

**Download into this folder:**

```bash
curl -L -o Mall_Customers.csv \
  "https://raw.githubusercontent.com/kennedykwangari/Mall-Customer-Segmentation-Data/master/Mall_Customers.csv"
```

Expected columns: `CustomerID`, `Gender`, `Age`, `Annual Income (k$)`, `Spending Score (1-100)`
